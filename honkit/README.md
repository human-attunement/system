# HAS Documentation

Human Attunement System（HAS）のドキュメント

HASはフレームワークでもメソッドでもない。  
人間の状態（State）が壊れ始めたときに、
選択が引き受けられなくなる連鎖を
遮断するためのセーフティーシステムです。

ただ自分や他者との関係の「状態」を見続けることで
その状態が壊れないようにします。

---

人は、以下の条件が揃うとき、  
自分の人生を「**自分のもの**」として引き受けられることができます。

- 自分の**内側から立ち上がったもの**である
- 結果の責任が**自分に属している**
- 侵されずに**存在できる場**がある
- **選ぶ／選ばない自由**が残っている

HASは、この4つが壊されないことだけを守ります。

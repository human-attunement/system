# HAS Public Relations Protocol
**HAS広報プロトコル：設置と境界の作法**

## 1. Core Philosophy
**「拡声器」ではなく「展示台」として使う。**

* 広めるためではなく、探している人が見つけられるように「置く」。
* 成功指標は「拡散数（Buzz）」ではなく「誤解の少なさ（Accuracy）」である。
* HASは、ユーザーの購買意欲や欠乏感を刺激してアクセスさせるものではない。

## 2. Posting Rules (Do's & Don'ts)

| Category | Action | Example / Mindset |
| :--- | :--- | :--- |
| **Do (やる)** | **事実と定義を書く** | 「これはXXなシステムです」 |
| | **限界と不向きを書く** | 「XXな人には役に立ちません」 |
| | **離脱の自由を保証する** | 「読まなくていいです」「スルーしてください」 |
| **Don't (やらない)** | **変化を約束する** | ❌「人生が変わります」「救われます」 |
| | **正しさを主張する** | ❌「今の時代に必須」「学ぶべき」 |
| | **扇動・共感を狙う** | ❌「拡散希望」「ピンときた人はRT」 |

## 3. The "Anti-Clickbait" Check
投稿ボタンを押す直前に、以下の問いにイエスと答えられるか確認する。

> **「この投稿を見た人は、『読まない自由』を完全に保持しているか？」**

* **No:** 不安を煽っている、正義感を刺激している、欠乏感を突いている。
    * 👉 **Reject (投稿禁止)**
* **Yes:** 読むか読まないかが、完全に相手の恣意に委ねられている。
    * 👉 **Post (設置)**

## 4. The Exit Clause (離脱条項)
すべてのコンテンツ、すべての接点の末尾に、以下のスタンスを明示する。

> **"If not now, forget it."**
> 今、必要でなければ、忘れてください。
> 必要になった時、また戻ってきてください。
